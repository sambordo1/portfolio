Python Boto3 Script for AWS IPs

Project Description
A python script that used boto3 library to take private ip addresses from aws and update the company ip addresses document

  • When running the script, the user provided a JSON input file, a csv output file, and AWS credential profile as arguments

  • The JSON input file included the company employees, ips used by each employee, and other information about employees

  • The script performs error checking on all these arguments (including whether the file exists and whether it is empty or not)

  • After performing checks, the script uses the boto3 python library and the AWS credentials provided to access the users AWS account and get the active private ip addresses used by the company

  • These ip addreses included EC2 instances, RDS Endpoints, and Redshift Endpoints.

  • The script then parses through the JSON file you gave it and takes all the ips out, then it compares them to the active ips from AWS

Skills Used:

  • Python
  • AWS CLI
  • Boto3 Library
  • Understanding of AWS
